# CRAB — варианты редизайна дашборда позиций

Интерактивная страница с вариантами нового дизайна страницы «Позиции» сервиса мониторинга CRAB.

**Смотреть: https://serjnsk.github.io/crab-dashboard-design/**

- Переключение вариантов — панель внизу страницы или стрелки ← →
- Светлая/тёмная тема — кнопка в той же панели
- Клик по ячейке позиции — снапшот ТОП-10 выдачи
- Наведение на «Позиции» в сайдбаре — список мониторингов

## Файлы

- `index.html` — страница для GitHub Pages (копия standalone)
- `crab-dashboard-standalone.html` — автономный файл для раздачи напрямую
- `crab-dashboard-variants.html` — рабочий файл (фрагмент для Claude Artifacts)
- `FRONTEND.md` — README для верстальщика: токены, пороги адаптивности, поведение компонентов, что в прототипе демо-специфично
